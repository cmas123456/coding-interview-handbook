# Blind 75 — Blind Drill

> Same 75 problems, **pattern labels removed**, shuffled within each difficulty band. Use this to drill pattern recognition on the FAANG-canonical set.

!!! tip "How to drill"
    1. Pick a problem. **Don't look at neighbors** — they are shuffled so adjacency tells you nothing.
    2. Open it on LeetCode. Read only the first paragraph + examples.
    3. Say out loud: which pattern, and why?
    4. If stuck, jump to **[Pattern Recognition](pattern-recognition.md)** — not to a specific pattern page.
    5. Check the box when you've recognized + solved (or recognized + understood the solution).
    6. Need the answer? See **[Blind 75 — By Pattern](blind75-by-pattern.md)** for the pattern labels.
