# Practice — Blind Drill

> Problems from every pattern page, with the pattern label stripped. Use this to practice recognition — read the title, predict the pattern, then click through.

!!! tip "How to drill"
    1. Pick a problem. **Don't** scroll to see what's near it (consecutive problems are NOT the same pattern — they're shuffled).
    2. Open it on LeetCode. Read **only the first paragraph + examples**.
    3. Say out loud: which pattern, and why?
    4. If stuck, jump to the [Pattern Recognition](pattern-recognition.md) decision tree — not to a specific pattern page.
    5. Check the box when you've recognized + solved (or recognized + understood the solution).
